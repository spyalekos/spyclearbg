# SpyClearBG

Μια premium εφαρμογή γραφικού περιβάλλοντος (GUI) για την αφαίρεση φόντου από φωτογραφίες με χρήση Τεχνητής Νοημοσύνης (rembg) και του Flet framework.

## Χαρακτηριστικά

- **Λειτουργία βάσει φακέλου:** Επιλέξτε έναν φάκελο και η εφαρμογή θα εντοπίσει αυτόματα όλες τις υποστηριζόμενες εικόνες (`.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp`).
- **Προεπισκόπηση side-by-side:** Δείτε την αρχική εικόνα και το επεξεργασμένο αποτέλεσμα δίπλα-δίπλα, με προσαρμοσμένο φόντο σκακιέρας που ταιριάζει με το θέμα της εφαρμογής.
- **Πληθώρα ρυθμίσεων rembg:**
  - Επιλογή μοντέλου AI (Standard, Fast, Human Segmenter, Cloth, Products/Siluela, ISNet General/Anime).
  - Ενεργοποίηση και παραμετροποίηση **Alpha Matting** (Foreground/Background thresholds, Erode size) για βέλτιστα αποτελέσματα σε λεπτομέρειες όπως μαλλιά ή φτερά.
  - Επιλογή παραγωγής **μόνο μάσκας** (ασπρόμαυρη).
  - **Μετα-επεξεργασία (Post-processing)** της μάσκας για πιο λεία άκρα.
  - Δυνατότητα ορισμού **προσαρμοσμένου χρώματος φόντου** (R, G, B, A) αντί για διαφανές.
- **Επεξεργασία με ένα κλικ:** 
  - Αφαίρεση φόντου από μία μεμονωμένη εικόνα άμεσα από τη λίστα ή το preview.
  - **Ομαδική επεξεργασία (Bulk Processing)** για όλες τις επιλεγμένες εικόνες μέσω checkboxes.
- **Ασύγχρονη Εκτέλεση (Multithreading):** Η επεξεργασία γίνεται σε background threads ώστε το γραφικό περιβάλλον να παραμένει πλήρως ανταποκρίσιμο χωρίς να παγώνει.
- **Διαχείριση Αρχείων:** Δυνατότητα ανοίγματος της αρχικής εικόνας ή του αποτελέσματος στο λειτουργικό σύστημα (Explorer) και διαγραφής του αποτελέσματος απευθείας από την εφαρμογή.
- **Premium Σχεδίαση:** Σκοτεινό θέμα (Dark green theme) σύμφωνα με τις προδιαγραφές, με χρήση CupertinoIcons και glassmorphism εφέ.

## Εγκατάσταση & Εκτέλεση

Η εφαρμογή απαιτεί Python 3.10 και διαχείριση μέσω `uv`.

1. Βεβαιωθείτε ότι βρίσκεστε στο σωστό περιβάλλον (venv) και έχετε εγκαταστήσει τα απαραίτητα πακέτα.
2. Τρέξτε την εφαρμογή με την ακόλουθη εντολή:

```bash
uv run python main.py
```

## Αποθήκευση Αποτελεσμάτων

Τα αποτελέσματα αποθηκεύονται αυτόματα στον ίδιο φάκελο με την αρχική εικόνα, προσθέτοντας την κατάληξη `_nobg.png` (ή την προσαρμοσμένη κατάληξη που ορίζετε στις ρυθμίσεις).
