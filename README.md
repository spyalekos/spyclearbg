# SpyClearBG 🖼️

**Αυτόματη Αφαίρεση Φόντου από Φωτογραφίες με Τεχνητή Νοημοσύνη**

Μια premium desktop εφαρμογή γραφικού περιβάλλοντος (GUI) που αφαιρεί αυτόματα το φόντο από εικόνες, χρησιμοποιώντας μοντέλα Τεχνητής Νοημοσύνης μέσω της βιβλιοθήκης [rembg](https://github.com/danielgatis/rembg) και το framework [Flet](https://flet.dev/).

## Χαρακτηριστικά

- **Λειτουργία βάσει φακέλου:** Επιλέξτε έναν φάκελο και η εφαρμογή εντοπίζει αυτόματα όλες τις υποστηριζόμενες εικόνες (`.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp`).
- **Προεπισκόπηση side-by-side:** Δείτε την αρχική εικόνα και το επεξεργασμένο αποτέλεσμα δίπλα-δίπλα σε πραγματικό χρόνο.
- **Πληθώρα μοντέλων AI:**
  - `u2net` — Γενικής χρήσης (προεπιλογή)
  - `u2netp` — Ελαφρύ/γρήγορο, για συσκευές με χαμηλούς πόρους
  - `u2net_human_seg` — Εξειδικευμένο για ανθρώπινες φιγούρες
  - `u2net_cloth_seg` — Εξειδικευμένο για ρούχα και ενδύματα
  - `siluela` — Σιλουέτες και προϊόντα
  - `isnet-general-use` — Νέο μοντέλο υψηλής ακρίβειας
  - `isnet-anime` — Εξειδικευμένο για anime / σκίτσα
- **Alpha Matting:** Ρύθμιση Foreground/Background thresholds και Erode size για βέλτιστα αποτελέσματα σε λεπτομέρειες (μαλλιά, φτερά κλπ.).
- **Μόνο μάσκα (Only Mask):** Παραγωγή ασπρόμαυρης μάσκας αντί για διαφανές αποτέλεσμα.
- **Μετα-επεξεργασία (Post-processing):** Λείανση ακρών της μάσκας.
- **Προσαρμοσμένο χρώμα φόντου:** Ορισμός R, G, B, A αντί για διαφανές.
- **Ομαδική επεξεργασία (Bulk Processing):** Επιλέξτε πολλαπλές εικόνες με checkboxes και αφαιρέστε το φόντο μαζικά.
- **Ασύγχρονη εκτέλεση:** Η επεξεργασία γίνεται σε background threads — το UI παραμένει πλήρως ανταποκρίσιμο.
- **Διαχείριση αρχείων:** Άνοιγμα αρχικής εικόνας ή αποτελέσματος στον Explorer, διαγραφή αποτελέσματος.
- **Premium σχεδίαση:** Σκοτεινό πράσινο θέμα (Dark green theme) με glassmorphism εφέ.

## Εγκατάσταση & Εκτέλεση

### Απαιτήσεις
- Python ≥ 3.10
- [uv](https://docs.astral.sh/uv/) (package manager)

### Από τον πηγαίο κώδικα
```bash
# Κλωνοποίηση του αποθετηρίου
git clone https://github.com/spyalekos/spyclearbg.git
cd spyclearbg

# Εγκατάσταση εξαρτήσεων
uv sync

# Εκτέλεση
uv run python main.py
```

### Εκτελέσιμο (Windows)
Κατεβάστε το τελευταίο `.exe` από τη σελίδα [Releases](https://github.com/spyalekos/spyclearbg/releases) και τρέξτε το απευθείας — δεν απαιτείται εγκατάσταση Python.

## Αποθήκευση Αποτελεσμάτων

Τα αποτελέσματα αποθηκεύονται αυτόματα στον ίδιο φάκελο με την αρχική εικόνα, προσθέτοντας την κατάληξη `_nobg.png` (ή την προσαρμοσμένη κατάληξη που ορίζετε στις ρυθμίσεις).

## Δημιουργία Εκτελέσιμου

```bash
uv run pyinstaller SpyClearBG.spec
```

Το εκτελέσιμο δημιουργείται στον φάκελο `dist/SpyClearBG.exe`.
